# Test CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a `tdf-iroh-s3-test` CLI binary for manual QA testing of deployed tdf-iroh-s3 nodes — creating TDFs, pushing blobs over Iroh, fetching blobs by hash, and sending invalid payloads.

**Architecture:** A second binary target in the same Cargo workspace, sharing the existing iroh/opentdf dependencies. Uses clap subcommands for `create-tdf`, `push`, `push-raw`, and `fetch`. Connects to a remote tdf-iroh-s3 node via Iroh's QUIC protocol using the node's Endpoint ID.

**Tech Stack:** Rust, iroh 0.97, iroh-blobs 0.99, opentdf (opentdf-rs), clap 4, tokio, blake3

use anyhow::{Context, Result};
use std::path::Path;

/// Create TDF bytes in memory with the given attribute and payload data.
pub fn create_tdf_bytes(attribute_fqn: &str, data: &[u8]) -> Result<Vec<u8>> {
    use opentdf::prelude::*;

    let mut builder = PolicyBuilder::new()
        .id_auto()
        .dissemination(["test@tdf-iroh-s3"]);

    if !attribute_fqn.is_empty() {
        builder = builder
            .attribute_fqn(attribute_fqn)
            .context("Invalid attribute FQN")?;
    }

    let policy = builder.build().context("Failed to build policy")?;

    let tdf_bytes = Tdf::encrypt(data)
        .kas_url("https://kas.example.com")
        .policy(policy)
        .to_bytes()
        .context("Failed to create TDF")?;

    Ok(tdf_bytes)
}

/// Create a TDF file on disk.
pub fn create_tdf_file(attribute_fqn: &str, data: &[u8], output_path: &Path) -> Result<()> {
    let tdf_bytes = create_tdf_bytes(attribute_fqn, data)?;
    std::fs::write(output_path, &tdf_bytes)
        .with_context(|| format!("Failed to write TDF to {}", output_path.display()))?;

    let hash = blake3::hash(&tdf_bytes);
    println!("Created TDF: {}", output_path.display());
    println!("BLAKE3 hash: {}", hash.to_hex());
    println!("Size: {} bytes", tdf_bytes.len());

    Ok(())
}
```

use std::path::PathBuf;

#[derive(Parser)]
#[command(name = "tdf-iroh-s3-test", about = "Test CLI for tdf-iroh-s3 nodes")]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Create a TDF file locally
    CreateTdf {
        /// Attribute FQN to include in the TDF policy
        #[arg(short, long)]
        attribute: String,

        /// Payload data (string)
        #[arg(short, long, default_value = "test payload")]
        data: String,

        /// Output file path
        #[arg(short, long)]
        output: PathBuf,
    },
}

#[tokio::main]
async fn main() -> Result<()> {
    let cli = Cli::parse();

    match cli.command {
        Commands::CreateTdf {
            attribute,
            data,
            output,
        } => {
            tdf_iroh_s3::test_cli::create_tdf::create_tdf_file(
                &attribute,
                data.as_bytes(),
                &output,
            )?;
        }
    }

    Ok(())
}
```

use anyhow::{Context, Result};
use iroh::endpoint::presets;
use iroh::{Endpoint, EndpointId};
use iroh_blobs::store::mem::MemStore;
use iroh_blobs::BlobsProtocol;
use iroh_blobs::Hash;
use std::net::Ipv4Addr;
use std::str::FromStr;

/// Parse an Endpoint ID string into an EndpointId.
pub fn parse_endpoint_id(s: &str) -> Result<EndpointId> {
    EndpointId::from_str(s).context("Invalid endpoint ID")
}

/// An Iroh client that can push and fetch blobs from a remote node.
pub struct IrohTestClient {
    endpoint: Endpoint,
    store: MemStore,
}

impl IrohTestClient {
    /// Create a new test client with an ephemeral identity.
    pub async fn new() -> Result<Self> {
        let store = MemStore::new();

        let endpoint = Endpoint::builder(presets::N0)
            .bind_addr((Ipv4Addr::UNSPECIFIED, 0u16))
            .context("Invalid bind address")?
            .bind()
            .await
            .context("Failed to bind client endpoint")?;

        endpoint.online().await;

        // Register blob protocol so we can add blobs locally
        let _blobs = BlobsProtocol::new(&store, None);

        Ok(Self { endpoint, store })
    }

    /// Add bytes to the local store and return the BLAKE3 hash.
    pub async fn add_bytes(&self, data: &[u8]) -> Result<Hash> {
        let tag = self.store.add_slice(data).await?;
        Ok(*tag.hash())
    }

    /// Push a blob to a remote node by its endpoint ID.
    /// Uses iroh-blobs download protocol in reverse: the remote node
    /// fetches from us after we advertise the blob.
    pub async fn push_to_node(&self, node_id: EndpointId, data: &[u8]) -> Result<Hash> {
        let hash = self.add_bytes(data).await?;

        // Connect to the remote node and send the blob
        let addr = iroh::EndpointAddr::from_parts(node_id, None, &[]);
        let conn = self
            .endpoint
            .connect(addr, iroh_blobs::ALPN)
            .await
            .context("Failed to connect to remote node")?;

        // Use the blobs protocol to push data
        iroh_blobs::provider::send_blob(&conn, &self.store, hash)
            .await
            .context("Failed to send blob to remote node")?;

        Ok(hash)
    }

    /// Fetch a blob from a remote node by hash.
    pub async fn fetch_from_node(&self, node_id: EndpointId, hash: Hash) -> Result<bytes::Bytes> {
        let addr = iroh::EndpointAddr::from_parts(node_id, None, &[]);
        let conn = self
            .endpoint
            .connect(addr, iroh_blobs::ALPN)
            .await
            .context("Failed to connect to remote node")?;

        // Request the blob from the remote node
        let data = iroh_blobs::get::get_blob(&conn, hash)
            .await
            .context("Failed to fetch blob from remote node")?;

        Ok(data)
    }

    /// Shutdown the client.
    pub async fn shutdown(self) -> Result<()> {
        self.endpoint.close().await;
        Ok(())
    }
}
```

**IMPORTANT NOTE:** The exact iroh-blobs client API for sending/fetching blobs may differ. The functions `iroh_blobs::provider::send_blob` and `iroh_blobs::get::get_blob` are approximations based on the crate structure. During implementation, explore the actual `iroh_blobs` API:
- Check `iroh_blobs::provider` module for sending blobs over a connection
- Check `iroh_blobs::get` module for fetching blobs
- The store's `.add_slice()` returns a tag with a `.hash()` method — verify the exact return type
- Connection establishment uses `endpoint.connect(addr, ALPN)` — verify `EndpointAddr::from_parts` signature

use anyhow::{Context, Result};
use iroh::EndpointId;
use iroh_blobs::Hash;
use std::path::Path;

use super::iroh_client::IrohTestClient;

/// Fetch a blob from a remote node by its BLAKE3 hash.
pub async fn fetch_blob(
    node_id: EndpointId,
    hash_hex: &str,
    output_path: Option<&Path>,
) -> Result<()> {
    let hash = Hash::from_hex(hash_hex).context("Invalid BLAKE3 hash hex string")?;

    let client = IrohTestClient::new()
        .await
        .context("Failed to create Iroh client")?;

    println!("Connecting to node {}...", node_id);
    println!("Fetching blob {}...", hash_hex);

    let data = client
        .fetch_from_node(node_id, hash)
        .await
        .context("Failed to fetch blob")?;

    println!("Fetched {} bytes", data.len());

    // Verify hash
    let computed_hash = blake3::hash(&data);
    if computed_hash.as_bytes() != hash.as_bytes() {
        anyhow::bail!(
            "Hash mismatch! Expected {} but got {}",
            hash_hex,
            computed_hash.to_hex()
        );
    }
    println!("Hash verified: {}", hash_hex);

    if let Some(path) = output_path {
        std::fs::write(path, &data)
            .with_context(|| format!("Failed to write to {}", path.display()))?;
        println!("Written to {}", path.display());
    }

    client.shutdown().await?;
    Ok(())
}
```
